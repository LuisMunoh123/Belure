% ABSTRACT
%----------------------------------------------------------

\begin{abstract}
En este informe se presenta un estudio comparativo de la complejidad temporal de algoritmos iterativos de búsqueda y ordenamiento aplicados a una base de datos ficticia de deportistas. Los experimentos se llevaron a cabo sobre conjuntos de datos de hasta 7000 elementos en tres escenarios de distribución distintos (Elementos previamente ordenados, en orden inverso y en orden aleatorio).

Los resultados demostraron una correspondencia significativa entre el análisis teórico de complejidad temporal y el comportamiento experimental observado.
\end{abstract}
