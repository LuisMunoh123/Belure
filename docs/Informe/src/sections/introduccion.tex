\section{Introducción}\label{sec:introduccion}

El diseño de algoritmos tiene como objetivo encontrar soluciones a problemas de manera eficiente teniendo en cuenta las limitaciones de tiempo y memoria que existen dentro de los sistemas informáticos.

Dos de los problemas más comunes en el día a día de la computación son el \textbf{ordenamiento} de datos y la \textbf{búsqueda} de información dentro de estructuras de datos; ambos problemas se pueden afrontar desde distintas miradas: algoritmos iterativos, recursivos, uso de estructuras de datos especiales, etc.

El presente informe aborda un problema de ordenamiento y búsqueda dentro de una base de datos ficticia de deportistas (generada proceduralmente) con el fin de exponer algunas de las soluciones más conocidas a estos problemas (\textit{Swap sort}, \textit{Insertion sort}, \textit{Selection sort}, \textit{Cocktail shaker sort}, \textit{búsqueda secuencial} y \textit{búsqueda binaria}), analizando su comportamiento a nivel teórico y experimental.
